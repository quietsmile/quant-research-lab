阶段 1 ｜ 技能与工具搭建（约 2–3 个月）

要学（核心课程，按重要性排序）：


量化研究方法论（最重要、最容易被跳过）： 过拟合、多重检验、样本内/样本外、walk-forward 滚动验证、前视偏差、幸存者偏差。这一块决定你会不会骗自己。
Python 数据栈： pandas、numpy，把数据"拿进来、清干净、跑统计"。
统计基础： 收益率/波动率/夏普/最大回撤、相关性、简单回归。
交易成本与 A 股摩擦建模： 把印花税、佣金、滑点、T+1、涨跌停撮合假设写进回测。


要用的工具：


数据：AKShare / Tushare（免费起步，够做中低频回测）
回测：backtrader / vnpy，或平台（聚宽 / 米筐 / 掘金）
让 AI 当生产力工具：帮你搭数据管道、写回测代码、查 bug、记研究日志


要做： 端到端跑通一个"玩具策略"：数据 → 处理 → 信号 → 带真实成本的回测。

通关标准（Gate 1）： 你能独立把一个策略从数据跑到带成本的回测，并能说清它为什么可能是过拟合。
