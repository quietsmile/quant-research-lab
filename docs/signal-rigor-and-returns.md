# 信号严谨性与收益：能挣多少、能挣到吗、为什么别人更高

本文记录对 ML 价量信号「可落地性」与「收益归因」的系统调查。结论一句话：**信号的横截面区分度真实，但扣掉 beta/成本/泄漏后，可落地的独立 alpha 很弱且低容量；别人 30–70% 的年化大头是泄漏 + 小盘 beta，而非真本事。**

所有脚本在 `examples/`，回测口径：liq1500 大中盘、含退市、2021–2026 OOS、walk-forward + 20 日 purge、扣双边成本。

---

## A. 乐观度拆解：30–70% 年化从哪来（`optimism_budget.py`）

同一个生产信号，逐步松绑严格项，看年化如何虚高：

| 松绑步骤 | 年化 | 夏普 | 回撤 |
|---|---|---|---|
| 0 严格基线（OOS/大中盘/真实撮合/扣成本/T+1开盘/hold10） | **+10%** | 0.72 | -31% |
| 1 +理想撮合（去涨跌停/停牌/ST） | +10% | 0.69 | -31% |
| 2 +去成本（毛收益） | +14% | 0.99 | -27% |
| 3 +日频调仓（hold=1） | +16% | 0.99 | -31% |
| 4 +下沉小盘（liq1500 内小市值半区） | +19% | 0.92 | -30% |
| 5 **+样本内信号（全期训练，泄漏）** | **+51%** | **1.82** | -30% |

**关键发现**：
- 去成本/日频/小盘（无泄漏的乐观/beta 选择）：+10% → +19%（×1.9）。
- **只加一步「样本内训练」（去掉 walk-forward）：+19% → +51%、夏普 0.92→1.82。** 同一信号、同一特征、同一模型，唯一区别是少了 walk-forward。
- 别人 30–70% 的年化，落点几乎精确命中「样本内泄漏」档（+51%）。

**结论**：那 30–70% 的大头是**样本内泄漏**，其次是小盘 beta + 无成本 + 日频。我们 +10% 的「低」，主要是严格的结果，不是模型差。真实可争取的合法空间在步骤 2–4 那 +10%→+19% 里的一部分（成本/持有期/特征/模型），但大中盘 + 扣成本 + 无泄漏下干净上 50% 物理上做不到。

## 已确立的方法论红线（前序调查，详见飞书子文档）

- **对冲归因**：标准做法 = 滚动 β 中性（PIT，禁用全样本 β = 未来函数）；更严谨 = Barra 多因子中性。滚动 β 中性后 N20 对中证500 残差 alpha 毛夏普 +0.84，扣 IC 贴水后仅 +0.25、N≥50 转负 → 半中证500 beta + 半弱 alpha。
- **多空 dollar-neutral 纸面夏普 ≠ 收益**（A 股个股不可做空），只作信号体检。
- **分层单调性**：top1% +1.90%/10日最强、无 roll-off → Top-20 低夏普是 breadth 问题；但可落地 P&L 显示扩 breadth 不救（edge 集中头部、衰减快过 √breadth）。
- **IC 提升技巧**：标签横截面 rank（+50%）、多视野集成（+6%）；样本外 RankIC 0.0477→0.0837，ICIR 0.515。
- **泄漏审计**：随机切分/label 时点/未来函数/purge 全干净。
